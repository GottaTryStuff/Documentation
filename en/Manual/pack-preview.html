    Preview packages

    Preview packages have not yet been verified to be safe to use with the current version of Unity, so you should only use them for testing and to provide feedback. At some point in the future, they might pass the verification requirements; however, they might also be deprecated instead. Because there is no guarantee for future support, you should not use any package that appears with the preview  indicator in production.For more information about the verified state and the lifecycle of packages, see Package states and lifecycle.

    For a list of stable packages verified for this release, see Verified packages.
